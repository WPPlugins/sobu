This is a plugin, it makes it possible to integrate a WooCommerce shop into the <cite>sobu</cite> network.

== Description ==

The prerequisite for this is that WooCommerce has been installed and a valid contract with the sobu platform has been signed. You can find further details at: https://www.sobu.ch/en/partner » 

The extension implements the following sobu features:

• Share function on the order confirmation page

• Share function in the order confirmation e-mail

• Share function in the order history section of the user account

• Users can track purchases (transfer of e-mails)

• Buddy page

• sobu info page

• direct links to products and categories

== Installation ==

The following procedure is recommended for the installation: 

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where to find the plugin settings page after the installation? =

In the WooCommerce Settings > Integration tab
